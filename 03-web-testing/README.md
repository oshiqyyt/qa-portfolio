# Веб-тестирование

Комплексное тестирование веб-приложений с использованием профессиональных инструментов.

## Что внутри

### Chrome DevTools (task_1.md)
**Объект:** [21-school.ru](https://21-school.ru/)

**Выполнено:**
- Изменение HTML/CSS «на лету» для демонстрации багов
- Поиск неиспользуемого CSS/JS (Coverage tab)
- Замер скорости загрузки при разных условиях (Fast 4G / Slow 4G / 300 kbit/s)
- Блокировка ресурсов для проверки устойчивости

### Кроссбраузерное тестирование (task_2.md)
**Объект:** [21-school.ru](https://21-school.ru/)

**Выполнено:**
- Тестирование 5–7 элементов интерфейса в 3 браузерах (Chrome, Firefox, Edge)
- Выявление и документирование отличий
- Анализ технических причин расхождений (движки браузеров)
- Рекомендации разработчикам по обеспечению кроссбраузерной совместимости


### UI/UX-тестирование (task_3.md)
**Объект:** Макет в Figma vs реализация

**Выполнено:**
- Сравнение макета с реализованным сайтом
- 3 баг-репорта на визуальные несоответствия
- Проверка шрифтов, отступов, цветов, размеров элементов

## Применённые навыки

- **Chrome DevTools:** Elements, Network, Coverage, Application
- **Кроссбраузерность:** понимание различий движков (Blink, Gecko, WebKit)
- **HTTP/HTTPS:** анализ кодов состояния, работа с сертификатами
- **Cookies:** анализ флагов безопасности (Secure, HttpOnly, SameSite)
- **UI/UX:** сравнение с макетами, валидация HTML (W3C)
