# Тестовые артефакты

В этом разделе представлены примеры тестовой документации, созданной в рамках проекта QA2 Школы 21.

## Что внутри

### Тест-кейсы (task_1.md)
**Объект:** [saucedemo.com](https://www.saucedemo.com/)

**Выполнено:**
- Позитивный тест-кейс: авторизация с валидными данными
- Негативный тест-кейс: проверка обработки невалидных данных
- Приложены скриншоты процесса тестирования

### Чек-лист (task_2.md)
**Объект:** Раздел «Как мы учим» сайта Школы 21

**Выполнено:**
- 5+ проверок основного функционала
- Краткое описание каждой проверки
- Тестирование по чит-листу (расширенные проверки)

### Тест-план (task_3.md)
**Объект:** Раздел «Корпоративное обучение» сайта Школы 21

**Выполнено:**
- 8+ пунктов тест-плана с описанием
- Адаптация под ГОСТ Р 56922-2016
- Описание стратегии, ресурсов, критериев начала/окончания тестирования

## Применённые навыки

- Структурирование тестовой документации
- Применение техник тест-дизайна
- Работа с требованиями и их анализ
- Оформление артефактов по стандартам
